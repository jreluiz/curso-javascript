# 🔗 Links Úteis — JavaScript

## 📖 Referência (para consultar, não para ler de capa a capa)

- [MDN Web Docs (em português)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — **a** documentação de JS/HTML/CSS. Quando pesquisar algo, prefira os resultados da MDN;
- [JavaScript.info (em português)](https://javascript.info/) — tutorial moderno e profundo, ótimo para revisar cada tema das aulas.

## 🎮 Prática interativa

- [freeCodeCamp (em português)](https://www.freecodecamp.org/portuguese/) — trilha de JS com centenas de exercícios corrigidos automaticamente;
- [Exercism — trilha JavaScript](https://exercism.org/tracks/javascript) — exercícios com mentoria da comunidade;
- [Codewars](https://www.codewars.com/) — desafios curtos por nível (comece pelos 8 kyu);
- [Flexbox Froggy (pt-br)](https://flexboxfroggy.com/#pt-br) — CSS Flexbox como jogo, para quem quiser caprichar no visual dos projetos.
